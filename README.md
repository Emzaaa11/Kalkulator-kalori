Program Perhitungan Kebutuhan Kalori Harian (BMR) ini digunakan untuk menghitung kebutuhan energi tubuh per hari berdasarkan rumus Basal Metabolic Rate (BMR). Program menerima input berupa jenis kelamin, berat badan, tinggi badan, dan usia pengguna menggunakan objek Scanner. Perhitungan BMR dilakukan menggunakan rumus yang berbeda untuk laki-laki dan perempuan, sehingga program menerapkan percabangan if untuk menentukan rumus mana yang digunakan. Selain itu, terdapat validasi input untuk memastikan pengguna hanya memasukkan jenis kelamin L atau P; jika input tidak sesuai, program akan menampilkan pesan kesalahan dan mengulangi proses menggunakan continue. Untuk membuat aplikasi lebih interaktif, program menggunakan loop while(true) sehingga pengguna dapat melakukan perhitungan berulang kali tanpa menjalankan ulang program. Setelah setiap perhitungan, pengguna diberi opsi untuk melanjutkan atau keluar dengan mengetik y atau n, dan perulangan dihentikan menggunakan break apabila pengguna memilih berhenti.
